# PRODIGY_SD_04
## Software Development Internship – Task 04
### Prodigy Infotech

## Task Title
Sudoku Solver Application

## Task Description
Developed a Java-based program that solves a given 9×9 Sudoku puzzle automatically. The program uses the backtracking algorithm to fill missing numbers while following all Sudoku rules.

## Features
- Solves standard 9×9 Sudoku puzzles
- Uses backtracking algorithm
- Validates row, column, and 3×3 sub-grid rules
- Displays unsolved and solved Sudoku grids

## Technology Used
- Java
- 2D Arrays
- Backtracking Algorithm

## How to Run the Program
1. Compile the program:
   javac SudokuSolver.java
2. Run the program:
   java SudokuSolver
3. View the unsolved and solved Sudoku output in the console

## Sample Output
Unsolved Sudoku grid is displayed first, followed by the solved Sudoku grid after applying the algorithm.
